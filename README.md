🔗https://mantine.dev/getting-started/

```
pnpm install @mantine/core @mantine/hooks
```

🔑 Generate UUID

Use the uuid package to generate unique IDs:

🔗 https://www.npmjs.com/package/uuid

Install:

```
pnpm install uuid

```

Usage example:

```
import { v4 as uuidv4 } from 'uuid';

const id = uuidv4();

```
